# Task 1 - Data Cleaning and Preprocessing

## Objective
To clean and preprocess the Titanic dataset for Machine Learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Loaded the Titanic dataset.
2. Checked missing values and data types.
3. Filled missing values using median and mode.
4. Converted categorical data into numerical form.
5. Standardized numerical features.
6. Visualized outliers using a boxplot.
7. Saved the cleaned dataset as Cleaned_Titanic.csv.
